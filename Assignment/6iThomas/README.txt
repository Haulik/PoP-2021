How to compile and run the code

Navigate via Terminal/CMD to the ../src folder. 
Use the command "fsharpc -a continuedFraction.fsi continuedFraction.fs" to combine a library.
Use the command "fsharpc -r continuedFraction.dll 4i1.fsx" to compile continuedFractionTest.fsx
Use the command "mono continuedFractionTest.exe" to run continuedFractionTest.exe
